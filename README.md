# Book-Publishing-Company
A web-based platform for managing and publishing books online.
Users can browse books, search by title, author or genre, and access detailed book information.

## Features
- User Registration and login
- View all books
- Search Books by Title, Author or Genre 
- Book Details page 
- Responsive Design

## Tech Stack
- Frontend: HTML, CSS, Javascript
- Backend: Node.js, Express.js
- Database: MongoDB
